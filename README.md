# Wallhaven 桌面助手 (Wallhaven Desktop Assistant)

一款基于 PyQt6 开发的 Wallhaven 壁纸搜索与批量下载工具，旨在提供与官网一致的筛选体验，并简化壁纸获取流程。

## 🌟 核心功能

- **深度筛选**: 支持关键词搜索，并提供分类（General/Anime/People）、纯净度（SFW/Sketchy/NSFW）、分辨率、比例及颜色筛选。
- **智能排序**: 支持按日期、相关性、随机、浏览量、收藏量、排行榜（Toplist）及热门（Hot）进行排序。
- **大图预览**: 点击缩略图即可开启高清大图预览模式，支持快捷键（左右方向键）切换图片。
- **批量下载**: 支持跨页面勾选图片，一键开启多线程批量下载，并实时显示下载进度。
- **一键壁纸**: 在预览或列表页点击“壁纸”按钮，即可直接将图片设为当前 Windows 桌面壁纸。
- **路径管理**: 自动记录上次选择的下载目录，并支持一键打开下载文件夹。
- **双模式切换**:
  - **游客模式**: 无需登录即可浏览 SFW/Sketchy 内容。
  - **登录模式**: 通过 API Key 登录后，可解锁 NSFW 内容（需在官网开启 18+ 选项）。

## 🚀 快速开始

1. **运行程序**: 双击运行 `Wallhaven.exe`。
2. **配置网络**: 由于 Wallhaven 服务器位于海外，**请确保您的电脑处于科学上网（VPN）环境**，否则将无法加载图片。
3. **搜索与筛选**: 在顶部工具栏输入关键词或调整筛选选项，点击“搜索/刷新”即可获取最新壁纸。
4. **下载壁纸**:
   - 点击“下载”按钮保存单张图片。
   - 勾选图片左上角的复选框，点击底部的“批量下载”进行批量获取。
5. **设置壁纸**: 点击“壁纸”按钮，程序会自动下载并将其设为桌面背景。

## 🔑 关于 API Key 与 NSFW

若要查看 NSFW 内容，请遵循以下步骤：
1. 访问 [Wallhaven 官网](https://wallhaven.cc/) 并登录账号。
2. 进入 `Account Settings` -> `Browsing`，开启 `NSFW` (18+) 选项。
3. 在 `Account Settings` -> `API Key` 中复制您的私有密钥。
4. 点击程序右上角的“🔑”图标，粘贴 API Key 并验证登录。

## 📂 存储说明

- **配置文件**: `config.json`（保存您的 API Key 和下载路径）。
- **默认下载目录**: 程序同级目录下的 `Wallhaven_Downloads` 文件夹。

## ⚠️ 注意事项

- **网络连接**: 必须开启科学上网工具。
- **SSL 证书**: 程序已针对 EXE 环境优化，自动处理 SSL 验证问题，无需手动安装证书。
- **临时文件**: 登录/注销采用动态刷新机制，不会产生多余的临时进程或文件锁定。
