# Wallhaven 桌面助手

一款简洁高效的 Wallhaven 壁纸搜索与批量下载工具。

## ✨ 核心特性
- **无限滚动**: 开启无限加载模式，向下滚动自动加载更多图片。
- **全能筛选**: 支持分类、纯净度（含NSFW）、分辨率、比例及颜色筛选。
- **极速体验**: 采用连接池技术，缩略图加载飞快，无感浏览。
- **一键壁纸**: 预览时点击“壁纸”按钮，即可直接设为桌面背景。
- **批量下载**: 支持跨页多选，一键开启多线程批量下载。
- **智能记忆**: 自动记住 API Key 和下载路径，无需重复配置。
- **通用网络**: 自动识别系统代理，支持手动配置自定义代理端口。

## 🚀 快速使用
1. **启动**: 运行 `Wallhaven.exe`。
2. **网络**: 点击右上角 🌐 图标配置代理（默认跟随系统代理）。
3. **NSFW**: 点击右上角 🔑 图标输入 API Key 即可解锁 18+ 内容。
4. **下载**: 默认保存至程序同级的 `Wallhaven_Downloads` 目录。
